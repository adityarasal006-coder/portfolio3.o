# Cyber-Shinobi Portfolio — Aditya Rasal

> **"Logic × Creativity × Discipline"**

A master-level, deployment-ready portfolio built with **Cyber-Shinobi Noir** aesthetics — a fusion of Neo-Brutalist Anime UI and Mathematical Tech Mysticism.

---

## Features

| Feature | Description |
|---|---|
| **Cinematic Loader** | Animated matrix grid background with character-by-character text reveal |
| **Custom Cursor System** | Dual-layer cursor with glow, trail, and hover state detection |
| **WebGL Hero** | Three.js animated shader background with crimson/electric-blue noise |
| **Scroll Animations** | GSAP ScrollTrigger-powered reveals, parallax, and counters |
| **Autonomous Executor Demo** | Live terminal with typing effect, rotating core visualization, and real metrics |
| **Decision Matrix Demo** | Interactive network graph with simulated decision paths and confidence scoring |
| **Responsive Design** | Mobile-first with hamburger menu, fluid typography, and touch optimizations |
| **Audio System** | Web Audio API ambient drone with toggle control |

---

## Tech Stack

- **Vanilla HTML5** + **CSS3** (custom properties, animations, grid/flexbox)
- **Vanilla JavaScript** (ES6+ classes, no frameworks)
- **Three.js** (WebGL shader background)
- **GSAP + ScrollTrigger** (scroll-driven animations)
- **Vite** (build tool and dev server)

## Deployment

### Option 1: Vercel (Recommended)
```bash
npm install -g vercel
vercel --prod
```

### Option 2: Netlify
Drag and drop the `dist/` folder into [Netlify Drop](https://app.netlify.com/drop).

### Option 3: GitHub Pages
```bash
npm run build
# Push dist/ folder to gh-pages branch
```

### Local Development
```bash
npm install
npm run dev
```
